# HR Analytics: Employee Attrition & Workforce Insights

## 📊 Project Overview
Human capital is the most valuable asset of any organization. This project analyzes employee data to uncover patterns related to attrition and performance. The goal is to provide HR managers with actionable insights to reduce turnover and enhance employee satisfaction.

## 🛠️ Tech Stack
* **Database:** MySQL (Data Cleaning & KPI calculation)
* **Analysis:** Python (Pandas & Seaborn for Correlation Matrix)
* **Visualization:** Power BI / Tableau
* **Key Metrics:** Attrition Rate, Average Salary, Years Since Last Promotion.

## 📂 Repository Structure
* `/SQL_Queries`: Scripts for calculating Attrition Rate and joining HR_1/HR_2 tables.
* `/Notebooks`: Python scripts for Exploratory Data Analysis (EDA).
* `/Dashboards`: PDF/Image exports of the HR Executive Dashboard.

## 🚀 Key Insights & Findings
1.  **Overtime Impact:** Employees working frequent overtime showed a 30% higher attrition rate compared to those who didn't.
2.  **Role Specifics:** Sales Representatives and Junior Technicians have the highest turnover, suggesting a need for better onboarding or incentive structures.
3.  **The "2-Year" Mark:** A significant spike in attrition occurs at the 2nd year of tenure, often correlating with a lack of recent promotions.

## 📈 How to Use
1.  Run the SQL scripts in `/SQL_Queries` to prepare the reporting tables.
2.  Open the Power BI file (`.pbix`) to view the interactive attrition filters.
3.  Refer to the Python notebook for the statistical breakdown of employee satisfaction scores.
